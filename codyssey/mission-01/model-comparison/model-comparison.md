# LLM 모델 비교·선정 보고서

## 1. 비교 목적
업무 자동화 과업("메일 초안 작성")을 동일한 입력 프롬프트로 3개 이상의 LLM에 수행시켜, "느낌"이 아니라 평가 축과 점수 근거를 기준으로 실무에 채택할 최적 모델을 선정한다.

## 2. 비교에 사용한 동일 입력
3개 모델 모두 아래와 동일한 프롬프트를 입력했다 (전문은 [model-comparison-test-design.md](model-comparison-test-design.md) 참고).

- 업무 과업: 메일 초안 작성
- 목적: 외부 협력사에 프로젝트 1차 산출물 제출 일정이 1주일 지연됨을 알리고 양해를 구하기
- 톤: 정중하고 격식 있는 비즈니스 톤
- 금지어: "어쩔 수 없이", "사정상" 등 책임 회피 표현
- 필수 포함: 제목/인사말/본문(지연 사실+사과+새 일정)/마무리

## 4. 평가 축 및 점수표

평가 축 5개, 각 1~5점.

| 평가 축 | Claude Sonnet 4.6 | Gemini 3.5 Flash | GPT-5.5 (ChatGPT 무료) |
|---|---|---|---|
| ① 형식 준수 | 5 / 구조 누락 없이 완벽히 준수 | 5 / 구조 누락 없이 완벽히 준수 | 4 / 대상(협력사 담당자) 명시 누락 |
| ② 톤 적합성 | 5 / 정중하고 격식 있는 비즈니스 톤 | 5 / 정중하고 격식 있는 비즈니스 톤 | 3 / 다른 모델 대비 문맥이 단조롭고 짧음 |
| ③ 지시사항 반영 | 5 / 핵심 포인트 모두 반영 | 5 / 핵심 포인트 모두 반영 | 3 / 구조 일부 누락, 격식 면에서 아쉬움 |
| ④ 한국어 자연스러움 | 5 / 어색한 표현 없이 자연스러움 | 5 / 어색한 표현 없이 자연스러움 | 5 / 어색한 표현 없이 자연스러움 |
| ⑤ 추가 디테일 | 5 / 지시 없이도 제출 일정·프로젝트명 등 구체화 제안 | 0 / 추가 제안 없음(무료판 한계로 추정) | 0 / 추가 제안 없음(무료판 한계로 추정) |
| **합계** | **25점** | **20점** | **15점** |

## 5. 모델별 결과 요약

- **Claude Sonnet 4.6**: 5개 평가 축 전부 만점. 형식·톤·지시 반영이 정확했고, 별도 지시 없이도 "제출 일정/프로젝트명 등 구체적인 내용이 있으면 반영해주겠다"는 추가 제안까지 자발적으로 제시함.
- **Gemini 3.5 Flash**: 형식·톤·지시 반영·자연스러움 4개 축은 만점이나, 추가 디테일 제안은 없었음. 무료판의 한계로 추정됨.
- **GPT-5.5 (ChatGPT 무료)**: 한국어 자연스러움은 만점이나, 대상 명시 누락(형식 준수 4점), 문맥이 단조롭고 격식이 다소 부족(톤 3점), 지시사항 반영도 구조 일부 누락(3점)으로 상대적으로 낮은 점수.

## 6. 최종 선정 결론

**Claude Sonnet 4.6을 최종 모델로 선정한다.**

1. 5개 평가 축 모두 만점(25점)으로, 형식 준수·톤 적합성·지시사항 반영 면에서 동률 1위인 Gemini와 차이가 없었다.
2. 결정적으로 "추가 디테일" 축에서 Claude만 유일하게 만점을 받았다. 별도 지시 없이도 제출 일정·프로젝트명 등 빈 정보를 구체화하도록 자발적으로 제안한 점은, 실제 업무에서 반복적으로 메일 초안을 맡길 때 사용자의 추가 입력 부담을 줄여주는 실질적 장점이다.
3. GPT-5.5 (ChatGPT 무료)는 한국어 자연스러움은 동등했으나 형식·톤·지시반영 3개 축에서 낮은 점수를 받아, "정중하고 격식 있는 비즈니스 메일"이라는 과업 목적에는 상대적으로 덜 적합했다.
4. 따라서 본 업무 과업(메일 초안 작성)에는 Claude Sonnet 4.6이 가장 적합하다고 결론 내린다.
